简介 ​

LeafFall Engine(落叶引擎)

一款专注于支持ADV/VN(视觉小说)游戏的二维引擎

注意:

⚠正处于内测版，并没有完整项目功能。

  • 本项目为GNU AGPLv3协议许可证。(目前没有确定)

包含以下项目的研发在内: ​

  • Ress (Resources Square) : 资源储存格式。
  • ScnScript (SCeNario Script) : 一种基于场景的代码语言。
  • TinySystem (ADVN's TinySystem) : ADV/VN 的游戏系统，支持自定义UI等

如何贡献？

如果您是用户：欢迎向我们反馈问题或提供建议。

如果您是开发者：您可以在自己的分支上开发新功能，并将已通过测试的功能合并到主分支上。（内测时不开放此功能）

如何构建？

推荐使用Visual Studio 2022及以上版本并确保安装了C#工作负载，然后打开本项目解方案文件进行构建。
